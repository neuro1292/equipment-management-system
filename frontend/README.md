# 🏥 Equipment Management System

A comprehensive equipment management system for Nepalgunj Neuro Hospital to track medical equipment, warranties, exchange history, parts replacement, and gate passes.

## 📋 Features

- **Equipment Registration** - Add and manage all medical equipment
- **Warranty Tracking** - Track warranty status and expiry alerts
- **Exchange History** - Record equipment exchange history
- **Parts Replacement** - Track parts replacement and maintenance
- **Gate Pass Management** - Manage incoming/outgoing materials
- **Reports** - Generate CSV reports and analytics
- **Dashboard** - Real-time statistics and overview

## 🚀 Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Styling:** CSS3

## 📦 Installation

### Prerequisites
- Node.js (v14 or higher)
- MongoDB (local or Atlas)

### Backend Setup

```bash
cd backend
npm install
npm run dev
